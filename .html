How to export your search terms report from Google Ads (30 seconds)
  1. Go to Google Ads → Insights and reports → Search terms (or open any campaign and click Insights → Search terms).
  2. Set your date range — last 30, 60 or 90 days is usually best. More data = better patterns.
  3. Make sure these columns are visible: Search term, Campaign, Ad group, Clicks, Impr., Cost, Conversions, Conv. value. Add via the columns icon if missing. Including Campaign + Ad group is what unlocks per-campaign analysis.
  4. Click the download icon (top right) → CSV. Don't pick "Google Sheets" — CSV is what we need.
  5. Drop the file in below, or paste the contents. The tool handles the 2-row preamble Google sometimes adds.

Tip: Also works for Microsoft Ads exports (similar column names) and Search Terms Insights from Performance Max.
